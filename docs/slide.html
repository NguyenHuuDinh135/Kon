AI Agent

LangGraph ReAct Agent

Trợ lý AI với 7 công cụ chuyên biệt, trả lời bằng tiếng Việt

🧠 LLM Configuration

Primary: AWS Bedrock Claude Haiku 3.5

Fallback: Ollama qwen2.5 (7B, local)

Embeddings: Titan Embed v2 (1024-dim)

Framework: LangGraph with ReAct pattern

🔄 Autonomous Loop (OAPA)

Observe — Scan database for anomalies
Analyze — Identify high-risk customers
Plan — Generate campaign strategies
Act — Create draft campaigns + notifications

🛠 7 Agent Tools

query_database — Read-only SQL (50 row limit)
get_customer_profile — Full profile + churn risk
get_churn_risk_summary — Aggregate churn stats
get_product_recommendations — Collaborative filtering
get_revenue_insights — Monthly trends + forecast
suggest_campaign — Segment-based campaigns
search_similar_customers — pgvector cosine search
